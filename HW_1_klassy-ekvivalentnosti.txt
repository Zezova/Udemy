Задание:
У вас есть поле ввода имени. Мин. количество символов для ввода в поле - 2, макс. - 64. Ввод только букв.
Напишите все классы эквивалентности, которые необходимы для тестирования, а также значения из них, которые вам понадобятся для создания тестовой документации с учетом анализа граничных значений

Вопросы к этому заданию
1) Укажите классы эквивалентности
2) Укажите представителей из классов эквивалентности с учетом анализа граничных значений
_________________________________________
Решение:
1) Укажите классы эквивалентности
(-бесконечность; 1][2; 64][65; +бесконечности)
- Из них позитивные:
* [2; 64], буквы
- Негативные:
* (- беск., 1], буквы
* [65, + беск.], буквы
* числа
* спецсимволы
2) Укажите представителей из классов эквивалентности с учетом анализа граничных значений
- для позитивного тестирования:
* 1 буква
* 2 буквы
* 3 буквы
* 32 буквы (середина класса)
* 63 буквы
* 64 буквы
* 65 букв
- для негативного:
* 0 букв (пустое поле)
* 66 букв
* 123 (ввод цифр)
* ^&(%# (спецсимволы)
